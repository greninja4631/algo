Jenkinsサーバ起動（Docker）

docker start jenkins


URLのポート:8080 である。





Homebrewで検査するDockerコマンド

  docker build --no-cache -t my-ds-ci .


初期段階の検証コマンド↓


 docker build --no-cache --progress=plain -t algo-ci .

   エラーを分析し、異なるエラー内容でもエラーが発生したファイルが同名ファイルであれば、デザイン的に見やすい形で整理して、
   ファイルごとにエラー内容を記載して下さい。エラー内容自体は妥協せずに、長文でも良いのでエラー箇所を全て漏れなく完璧に記載して下さい。
